Experiência Criativa: Navegando na Computação

A matéria "Experiência Criativa: Navegando na Computação" oferece aos estudantes de Ciência da Computação na PUC-PR a oportunidade de explorar e experimentar com diferentes áreas da computação de maneira criativa, desenvolvendo habilidades práticas e ampliando sua compreensão do campo da tecnologia.

Fundamentos de Sistemas Ciberfísicos

a disciplina "Fundamentos de Sistemas Ciberfísicos" proporciona uma introdução aos conceitos e princípios essenciais relacionados à integração entre sistemas computacionais e físicos. Aprendemos sobre o uso de sensores, atuadores e protocolos de comunicação para projetar e desenvolver soluções interativas e inteligentes no mundo real.

Raciocínio Algorítmico

a disciplina "Raciocínio Algorítmico" ensina os fundamentos da resolução de problemas por meio de algoritmos. Aprendemos a analisar, projetar e implementar soluções eficientes usando estruturas de dados e algoritmos básicos, desenvolvendo habilidades essenciais para a programação.

Conhecimentos

HTML5

HTML5 é uma linguagem de marcação de hipertexto, que usamos para marcar os elementos da página e construí-la.

CSS3

CSS3 é uma linguagem de estilo usada para descrever a apresentação dos elementos da página os estilizando.

JavaScript

JavaScript é uma linguagem de programação interpretada usada no desenvolvimento web para descrever funcionalidades e comportamentos da página.

Python

Python é uma linguagem de programação versátil e poderosa, utilizada em uma ampla gama de aplicações, desde desenvolvimento web até análise de dados e inteligência artificial. Sua sintaxe legível e expressiva torna a codificação mais eficiente e acessível.
